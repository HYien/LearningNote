# 如何用脚手架创建项目
## 1.安装脚手架工具
```bash
npm i -g create-react-app
```
使用以上命令全局安装Create React App脚手架工具。

## 2.查看脚手架工具的版本号
```bash
create-react-app --version
```

## 3.使用脚手架工具创建项目
```bash
create-react-app react-demo
```

## 4.使用命令启动项目
```bash
npm start
```
